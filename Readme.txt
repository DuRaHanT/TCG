- Rule -

========================================================

1. 종족

- 종류 -
자연, 동물, 기계

2. 코스트

- 종류 -
동물, 기계, 자연, 무속성, 만능

- 최대치 -
10, 저장 불가

- 회복 방법 -
각 카드에 회복량이 정해져 있음, 무속성의 경우 매턴 2씩 회복

3. 덱 구성 방법

- 같은 이름의 카드는 최대 2장까지 들어 갈수있다.
- 덱 최대 매수는 총 30장이다.

4. 진행 방식

- 플레이어의 체력은 각각 5씩으로 직접공격 한번당 1씩 줄어들며 0이 먼저 되는쪽이 패배한다.
- 패에 소지할 수 있는 카드의 수는 최대 10장이다.
- 패에 있는 카드 2장을 버리면 코스트 하나를 만능코스트로 변경이 가능하다.
- 필드에 낼 수 있는 몬스터는 앞열 3마리, 후열 3마리 총 6마리이다.
- 앞열로 몬스터를 보내는 방법은 후열에서 소환한 후 다음턴부터 앞열에 자리가 있을경우 앞열로 이동을 하게된다.
- 앞열로 보낼경우 몬스터가 소환된 같은 세로열로 움직이는 것이므로 카드의 특성이 아닌이상 다른 칸으로는 이동이 불가능하다.
- 한턴에 사용할 수 있는 어빌리티 카드는 최대 2장까지이다.
- 앞열과 앞열이 싸울경우 선공권은 선공측이 가져간다.
- 선공은 첫턴에 드로우를 못한다.
- 앞열이 존재하지 않으면 상대 앞열에게 직접공격을 당한다.

A를 선공 B를 후공이라하면
A의 메인 - B의 메인 - 배틀페이즈
